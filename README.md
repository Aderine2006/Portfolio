## ✨ What Makes This Portfolio Unique?

Unlike standard templates, this platform is custom-engineered to mirror the aesthetic of world-class engineering teams (like Vercel and Linear).

*   **Immersive Interaction Strategy**: Every card, button, and metric reacts to the user's cursor movement with high-frequency "Spotlight" effects and reactive glassmorphism.
*   **Technical Workflow Visualization**: Complex AI projects (like the Vehicle Fleet Maintenance Agent) are presented with their actual technical architecture (n8n workflows) visible, proving technical depth at a glance.
*   **Quantifiable Success Metrics**: Every experience and achievement is backed by numbers—efficiency gains, student counts, and industry delivery rates.
*   **Performance-First Animations**: Powered by hardware-accelerated Framer Motion, ensuring 60fps transitions even on mobile devices.

## 📂 Portfolio Architecture (Sections)

### 1. **The Hero Experience**
*   **Dual-Rotating Rings**: Dynamic visual branding surrounding the profile avatar.
*   **Instant Resume Access**: A custom "Download CV" action integrated directly into the avatar section for recruiter convenience.
*   **Strategic Metrics**: Immediate visibility into core strengths (e.g., 6th Sem CSE, Community Leadership).

### 2. **Technical Skills**
*   **Categorized Domains**: Organized into Frontend, Backend, AI/Tools, and Core Languages.
*   **Dynamic Brand Logos**: Official icons that transition from subtle grayscale to vibrant brand colors on hover.
*   **Floating Motion**: Subtle "float" animations that give the section a living, breathing feel.

### 3. **Achievements & Milestones**
*   **Impact Metric Cards**: Large, bold numbers representing University-level wins and 500+ students mentored.
*   **Contextual Coloring**: Each achievement type has its own signature glow (Gold for Wins, Cyan for Education, Purple for Industry).

### 4. **Professional Experience**
*   **High-Contrast Timeline**: A clean, vertical flow of your professional journey.
*   **Sparkle Metrics**: Key achievements inside each role are highlighted with visual sparks.
*   **Integrated Certification**: Direct access to Internship Certificates (e.g., Python Developer Intern) directly from the job card.

### 5. **Selected Projects**
*   **Case-Study Focus**: Goes beyond "description" to explain the **Problem**, **Approach**, and **Impact**.
*   **Direct Repository Links**: One-click access to the source code on GitHub.

## 🚀 Deployment Workflow

The project is configured for **Next.js Static Export** to ensure maximum performance and security on Firebase.

### Build & Export
```bash
npm run build
```
*This triggers the static export defined in `next.config.ts`, generating optimized HTML/CSS/JS in the `out/` directory.*

### Deploy
```bash
firebase deploy
```
*Uploads the production bundle to the Global Firebase CDN.*
